####################################################################################################

This directory contains the VOM-documentation.


To view the documentation, open the index.html with your favorite browser.

####################################################################################################

If you want to add something to the documentation:
----------------------------------------------------------------------------------------------------
First (most common):

New version number and informations	- Update the version-txt.txt file
					 (contents will be displayed in VOM-version.html)

----------------------------------------------------------------------------------------------------
Second:

Updating existing html files		- Add your extras inside the relevant html files

Adding new html files 			- Create a new html file in the directory;
					- Link the file inside the navigation bar (navi-bar.html)
----------------------------------------------------------------------------------------------------
If you are new to html-coding, you can use the dummy.html file. This file contains the basic
html-elements that are used for this documentation.

Please try to use only the basic html elements. We want to keep it simple !

####################################################################################################
