The file read_nc_lsh_daily.f90 can be used for creating a dailyweather.prn for use in VOM. The use, the needed input and the resulting output are described in the file itself. As input files are meteorological data needed which are provided by the Land Surface Hydrology Research Group in the Department of Civil and Environmental Engineering at Princeton University. The data "Global Meteorological Forcing Dataset for land surface modeling" is freely available and can be downloaded via http://hydrology.princeton.edu/data.php
