Description of perc_cov.txt

The file "perc_cov.txt" is an optional input file for VOM, containing daily coverage data of the seasonal vegation in ascii format.

Fields

Dcum Day of the year Integer, starting with 1 on 1 January
Day Day of the month Integer
Month Month of the year Integer, starting with 1 on January
Year Year Integer, 4 digits
Relative coverage Fraction seasonal coverage - Decimal fraction between 0 and 1
