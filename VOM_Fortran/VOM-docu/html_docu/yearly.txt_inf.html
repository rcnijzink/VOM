Description of yearly.txt

The file "yearly.txt" is an output file from VOM, containing annual weather and flux data in ascii format for all years of the model run.

Fields

year year
rainyr yearly rainfall (in mm/a)
epanyr None
paryr Photosynthetically active Photon Flux Density (in mol m-2 a-1)
radyr Global solar irradiance (in MJ m-2 a-1)
vdyr Yearly average (of daily averaged) atmospheric vapour deficit (in mol mol-1)
etyr Yearly transpiration rate by vegetation (in m a-1)
esoilyr Yearly soil evaporation rate (in m a-1)
netassyr
gppyr

The first line contains the variable names.
